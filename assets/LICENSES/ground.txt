Processing note: the 4 grass-over-dirt tiles (tile_0000-0003) are real,
hand-drawn material-transition art — genuine variety across a tiled run, not
per-pixel random noise standing in for texture. tile_0004 (plain dirt, no
grass cap) is used for the underside/interior frame — a real, correct
material-transition signal (interior dirt has no grass), not a fabricated
edge shape. All 5 tiles are palette-remapped onto this project's locked dark
environment token set (#0a0a0a/#0f0f0f/#111111/#151515/#1a1a1a/#2a2a2a/#0f0f1a)
via a luminance-ramp quantization pass (see scripts/build-art-assets.py's
`_remap_luminance`) that preserves the source tiles' real light/dark structure
— raw RGB nearest-color matching against this narrow, near-black palette was
tested and rejected: it collapsed all 5 tiles to a near-flat single color,
destroying the material-transition detail. This is a derivative recolor of the
same CC0 source art, not new artwork; CC0 permits modification.
