Processing note: source poses are higher-resolution pixel-art character
illustrations than this project's tight 16x32 collider frame. Each pose is
cropped to its content bounding box, then ALL 5 poses are scaled by ONE shared
factor (derived from the tallest pose's content height) so the character does
not visibly grow/shrink between idle/run/jump states, resized with
nearest-neighbor only, bottom-aligned to the frame, and finally palette-remapped
onto this project's locked player token set (#e8e8e8/#d8d8d8/#909090/#444444 on
#0a0a0a) via a nearest-color quantization pass. This is a derivative crop/recolor
