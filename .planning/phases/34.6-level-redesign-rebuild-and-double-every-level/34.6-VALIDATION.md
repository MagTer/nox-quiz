## Sampling Rate

- **After every rebuilt descriptor:** Run `node scripts/validate-levels.mjs` (+ coin-reachability inside it)
- **After every plan wave:** Run the bidirectional `node scripts/browser-boot.mjs` driven-player pass + full gate suite
- **Before `/gsd-verify-work`:** Full suite green, zero HARD-FAILs across all 8 levels
- **Max feedback latency:** ~180 seconds
