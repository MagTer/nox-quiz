    The door panel emission (`add([sprite("door"), pos(d.x, d.y), "door-panel"])`) needs NO code change. All three invisible blocker `add([rect(...), ..., area(), body({ isStatic: true }), ...])` calls (doors, math-gates, enemies) must remain completely byte-unchanged — this is a visual-layer-only swap, per this phase's "geometry/collider frozen, visual pass only" discipline (same as Phase 32's terrain work).
