    Logic:

  <action>
    Inside `buildLevel`, add a helper `pickTopFrame(tx, runX, runW)` that returns the correct `ground.png` frame index:

    ```js
    function pickTopFrame(tx, runX, runW) {
      const isLeft = tx === runX;
      const isRight = tx + CONFIG.TILE_SIZE >= runX + runW;
      if (isLeft && isRight) return 0; // single-tile run
      if (isLeft) return 1;            // left edge
      if (isRight) return 3;           // right edge
      return 2;                        // center fill
    }
    ```

    For each visual floor tile inside the `g.floors` loop, add `frame(pickTopFrame(tx, run.x, run.w))` to the `sprite("ground")` component list. Do the same for each visual platform tile inside the `g.platforms` loop using `p.x`, `p.y`, and `p.w`.

    Do NOT change the merged colliders above the tile loops. Do NOT add `area()`/`body()` to the visual tiles. Do NOT introduce per-tile colliders. The `frame()` component must come after `sprite("ground")` in the component array.
  </action>
