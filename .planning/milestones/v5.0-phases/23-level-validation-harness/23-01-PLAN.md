<objective>
Measure the real jump envelope against the running engine and freeze it into a checked-in constant, replacing the flat, no-safety-factor closed-form formula Phase 22 already proved produces unsafe-margin false candidates.

This plan is the calibration step; Wave 2's reachability module (Plan 23-04) consumes its frozen output.
Output: `scripts/calibrate-jump-envelope.mjs` (a Playwright probe, run once) and `scripts/lib/jump-envelope.mjs` (the frozen, documented `JUMP_ENVELOPE` constant it produces).
</objective>
