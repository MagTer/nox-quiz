## Why (blocker found 2026-07-16, user full test-play of all 8 levels)

The Phase-34.6 rebuild produced structurally **CLONED** levels:
- **Even levels 02 / 04 / 06 have byte-identical platform layouts** — platform y-sequence
  `254,246,172,98,98,24,-50,-124,-198,-272` at nearly identical x's. 04 (town) and 06 (cemetery)
  are just level-02's "Rusted Spire" re-skinned.
- **Odd levels 01 / 03 / 05 are near-identical** arch+fork variations.
- Only **07 / 08** got genuinely distinct layout work.
- ALSO: long boring **"transport" sections** across levels — flat ground → jump a hole → flat
  ground → jump a hole, on repeat. No varied action.

**Root cause (my mistake):** the per-level executor dispatch prompts told each executor to
"match/exceed level-02's ambition" and handed 01/02 as the template → executors re-skinned the
template instead of inventing distinct levels. And there was NO automated distinctness/variety gate,
so it passed green and the prototype sign-off only ever saw 01/02.

**User decision (2026-07-16):** REDO ALL 8 with genuinely distinct shapes + no transport monotony
+ two new enforcing gates. (User picked "rethink the whole set (all 8)".)

## Build approach (do NOT repeat the clone mistake)

- **Design/build each level to its OWN unique spec. NEVER tell an executor to "match/reference
  level X" as a structural template** — that is exactly what caused the cloning. Give each level its
  own distinct skeleton.
- **Build on Opus executors** (per memory `level_design_needs_opus` — Sonnet builds flat levels).
- **Every level internally VARIED — no "transport":** never a long run of flat same-height
  floor→gap→floor→gap. Interleave climbs, descents, hazard-timing sequences, route forks, and
  set-pieces; the platforming "beat" must keep changing.
- **Keep (unchanged):** math density (1 door + 1 enemy + end gate); even levels (02/04/06/08) carry
  the math-skip key (`geometry.keys`, NO `geometry.locks`; key held → free clear +20 XP, else end
  math); the §8.5 raised bar; fall-stakes with close checkpoints (§8.5 rule 4); open-air (no
  ceilings/tunnels — always see the route); no timers / no game-over; ~2x length capped by the HARD
  perf-objects 650-tile budget (build to the ceiling).
- **Consider a HOLISTIC design pass first** (one Opus agent designs all 8 distinct + varied as a
  set, produces a beat-map per level) so distinctness is guaranteed cross-level, THEN build each —
  rather than 8 isolated executors (which is what cloned them). Recommended.
- 01/02/07/08: 02/08 mostly kept (trim transport); 01 → Bog Crossing (real change); 07 → Ramparts
  (real change, replaces staircase).
