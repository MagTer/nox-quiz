# Phase 34.6 Plan 05: Rebuild level-03 (calm town intro) Summary

**Rebuilt `src/levels/level-03.js` from scratch as a ~2x-length calm TOWN intro (goal.x 5120 -> 10280) with its own townscape signature — a mandatory 200px rooftop climb over a plaza pit, a visible low/high fork, a covered arcade awning, an optional 222px belltower (the game's first overlapping-tier ceilings, on optional routes only), and townsquare stepping stones — proven in-engine (zero HARD-FAIL, all 40 coins collected, door+enemy+alcove triggered, completes spawn->goal).**

## Design record (per the verification MANDATE)

| Property | Value |
|---|---|
| Climb (mandatory rooftop) | 200px (street y:320 -> rooftop peak RC y:120) — a notch more than level-01's 196px |
| Max vertical range | 222px (optional belltower top BT3 y:98) — mid odd-band (§8.5 ~150-250px) |
| Deliberate descent | rooftop RC -> RD -> down to F3 (mandatory); belltower drop-back (optional) |
| Fork / routes | 1 visible fork on F2: LOW road (street) vs HIGH road (RF1/RF2/RF3, 3 bonus coins), diverge + rejoin; no key (odd) |
| Checkpoints | 21 (one per rooftop-climb tier RA/RB/RC, before every spike, before the enemy, one per street stretch) |
| Signature | TOWNSCAPE — rooftops + a covered arcade + an optional belltower |
| Distinct from | level-01 (single swamp arch, no ceilings) and level-02 (tall swamp switchback spire + key); level-03 is a town, calmer, ceilings-appearing |
| Preserved | town biome, ~2x length, 1 door + 1 enemy + math end gate, mathGates: [], NO key, one secret alcove, open required path, explicit bounds (right hand-bumped 5200 -> 10380) |

## Accomplishments
- Rebuilt `src/levels/level-03.js` entirely (append-only suspended, §9.1): 13 street runs, 14 platforms (all `h:16`, all off-grid `y`), 40 coins, 4 spikes, 1 door, 1 enemy, `mathGates: []`, 1 secret alcove, 21 checkpoints, explicit `bounds` with `right` hand-bumped to 10380.
- Authored the town's distinct signature: a MANDATORY stepped rooftop climb (RA->RB->RC peak y:120 = 200px -> RD descent) over a 480px plaza pit, made mandatory because F3 starts beyond RB's reach (the exact level-01 arch trick, re-proven by the validator's own jump-envelope BFS).
- Introduced the game's FIRST overlapping-tier ceilings — but ONLY on optional routes: the covered arcade awning AR1 (a brief street duck-under) and the optional belltower BT1/BT2/BT3 (three overlapping tiers at rise 74, 26px headroom each, PASS) — satisfying §8's "L3-6 ceilings start appearing" while keeping the required path open/legible (§8.5 rule 6).
- Added a visible low/high fork (RF1/RF2/RF3, 3 bonus coins) and townsquare stepping stones (SS1/SS2) for beat-to-beat variety, with the mandatory 200px rooftop climb as the vertical centerpiece.
- Kept it a calm-but-harder-than-01 intro: 160px town-band gaps (up from level-01's 120/140), gentle 64-70px mandatory rises, only 4 spikes (each >=250px from both floor edges, clear of every platform footprint), and dense checkpoints so any plaza-pit fall costs seconds.
- Re-baselined the level-03 golden fixture in `scripts/smoke-progress.mjs`; only that block changed.
- `validate-levels.mjs` now reports zero HARD-FAIL for level-03, deleting its 3 previously-deferred Phase-34 coin fails — the project-wide HARD-FAIL count dropped 8 -> 5 (the remaining 5 are level-07's not-yet-rebuilt headroom fails, out of scope).

## Decisions Made
- Reused level-01's proven mandatory-arch shape as level-03's rooftop climb (same RA/RB/RC/RD skeleton, same "F3 beyond RB's reach" mandatory-peak trick) rather than inventing new climb physics — the risk was in the town character and the new overlapping tiers, not the arch.
- Put ALL overlapping-tier ceilings on OPTIONAL routes (arcade + belltower) so the required street never forces a cramped duck — the calm way to introduce town ceilings.
- The belltower descends by a safe drop back onto solid floor (no descent tiers, no pit), saving object-budget headroom and reading as a natural "climb-and-hop-down."
- 160px town-band gaps (vs level-01's 120/140) as the deliberate difficulty step: level-03 is the town INTRO but a rung up from level-01.

## Deviations from Plan
None - the level validated, drove, and audited clean on the first authored geometry (no in-engine defects surfaced this time; the >=250px spike-margin and mandatory-arch patterns inherited from 34.6-02/03 were applied from the start rather than rediscovered).

## Object budget
Terrain object count stayed under the 650 `OBJECT_BUDGET` (browser-boot's HARD `perf-objects` gate PASSED on both runs). Floors were kept lean and platforms modest specifically because level-03 is the longest level (goal 10280, longer than 01/02) — the budget was the main authoring constraint, managed by keeping the belltower descent a drop-back and the optional structures narrow.

## Issues Encountered
**Transient perf-fps load-contention flake, consistent with the documented 34.6-01/02/03 precedent.** Every `browser-boot.mjs` run flagged `perf-fps` for levels 01/02/03 (34-37 fps vs the 40 floor) on this dev machine — never a route/stall/death error on level-03 in either of two runs. One first run additionally showed a `far-end-unreachable` stall on level-04 (OLD, not-yet-rebuilt geometry, goal.x:6120) that did NOT reproduce on the second run — the same machine flakiness recorded in the prior summaries. A mid-iteration re-run also crashed at Playwright browser teardown (`browserContext.close`) under a `timeout` wrapper — a harness teardown race, not a level defect; the two clean runs are authoritative.
